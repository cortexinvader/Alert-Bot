🚀 AlertBot API

📋 Overview

AlertBot is a secure API for delivering alerts through multiple channels:

📧 Email 💬 Telegram 📱 Facebook Messenger

🔑 Authentication

All API requests require an API key sent in the header:

X-API-Key: your-api-key-here

📡 Send Message Endpoint

POST /send

Request Body:

{
  "channel": "email" | "telegram" | "facebook",
  "recipient": "address_or_id",
  "message": "Your message here"
}

Response:

{
  "status": "sent" | "failed",
  "details": "..."
}

💡 Examples

Email Example:

curl -X POST https://your-domain/send \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "channel": "email",
    "recipient": "user@example.com",
    "message": "This is a test alert"
  }'

Telegram Example:

curl -X POST https://your-domain/send \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "channel": "telegram",
    "recipient": "123456789",
    "message": "Alert from AlertBot"
  }'

Facebook Example:

curl -X POST https://your-domain/send \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "channel": "facebook",
    "recipient": "psid_here",
    "message": "Important notification"
  }'

🤖 Getting User IDs

Telegram:

Users can get their Telegram ID by:

  • Sending /getid command to your bot
  • Clicking "Get My ID" button in the bot

Facebook:

Users can get their PSID by:

  • Clicking "Get Started" on your Facebook page
  • Sending /getpsid command in chat
