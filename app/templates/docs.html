📋 Overview

AlertBot is a secure API for delivering alerts through multiple channels:

📧 Email 💬 Telegram 📱 Facebook Messenger

🔑 Authentication

All API requests require an API key sent in the header:

X-API-Key: your-api-key-here

📡 Send Message Endpoint

POST /send

Request Body:

{
  "channel": "email" | "telegram" | "facebook",
  "recipient": "address_or_id",
  "message": "Your message here"
}

Response:

{
  "status": "sent" | "failed",
  "details": "..."
}

🤖 Getting User IDs

Telegram:

Users can get their Telegram ID by interacting with Alert Bot:

  • Sending /getid command to the bot
  • Clicking "Get My ID" button in the bot

Facebook:

Users can get their PSID by interacting with the bot at Alert Bot :

  • Clicking "Get Started" on the Facebook page
  • OR Sending /getpsid command in chat
